# Richess
Richers is a cross-platform 3D chess game.<br />
Application has been written in C++ language.<br />
OpenGL and GLFW libraries have been used.<br />

## Build instructions

* Clone the Richess game repository.
* Move into the `src` directory.

* Run Richess.
